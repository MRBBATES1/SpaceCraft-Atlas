# SpaceCraft Atlas

SpaceCraft is currently in early access and, as it stands, the game offers very limited tools for tracking where resources are located across the universe. Finding what you need often comes down to luck and a lot of time spent jumping between systems blind. The SpaceCraft Atlas exists to fix that — a community-built reference that lets players go directly to where they need to be, cutting out the guesswork and getting them back to playing faster.

---

## What's in This Repo

This repository contains two browser-based tools and the map data that powers them:

- **SpaceCraft Atlas** — the viewer. Load a map for your server or quadrant and browse every sector, system, planet, and resource in one place. Full search and filtering included.
- **SpaceCraft Atlas Editor** — the editor. A template-driven tool for building and maintaining Atlas map files as you explore and discover new things in-game.
- **Atlas Maps/** — a collection of pre-built maps for available regions and quadrants, ready to load straight into the Atlas.
- **Templates/** — a blank map template to start from if you're building a new map from scratch.

For more detail on how each tool works, see their individual READMEs.

---

## A Note on Data Accuracy

The maps in this Atlas are built from information gathered and shared by players. Coverage is ongoing — not every system, planet, or resource will be documented yet, and some details may be incomplete as exploration continues. What you see reflects what the community has found and contributed so far.

---

## Want to Contribute?

If you've been exploring and want to add your findings, the Atlas Editor is the place to start. It walks you through logging new sectors, systems, planets, resources, and stations using simple templates — no JSON knowledge needed. When your map is ready, you can open a pull request to share it with the rest of the community. Contributors are credited within the app.

See the **Atlas Editor README** for full instructions, including file naming conventions and structure requirements.

---

## The Future of This Project

If Shiro Games improves the in-game interface in future updates to the point where a tool like this is no longer needed, this project will be retired. Until then, I'll continue maintaining and updating the Atlas as new information comes in.
